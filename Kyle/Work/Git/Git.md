### Git的由来
Linux内核开源项目参与者众多，为了方便维护工作，减少花在提交补丁和保存归档的繁琐事务上的时间，为了支持非线性开发模式；
-   2002年，开始使用分布式系统 BitKeeper 进行管理；
-   2005年，社区不能再**免费使用**BitKeeper（因为有人想破解BitKeeper的协议~）；
于是社区就结合 BitKeeper 的使用经验开发出了自己的版本系统Git；（最早的Git就是Linux之父Linus开发的）
